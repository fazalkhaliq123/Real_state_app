# Real_state_app
🏡 Real Estate Property Finder  A modern Real Estate Web App built with Next.js and the Bayut RapidAPI, allowing users to search and filter properties for rent and sale.  ✨ Features  🔍 Advanced Search &amp; Filters  Purpose (Rent / Sale)  Rent Frequency  Price Range (Min/Max)  Rooms &amp; Baths  Furnishing Type  Property Type  Location Search
